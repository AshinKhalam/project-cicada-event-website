CODE JAM

Confident you can code? How about some brainstorming questions to play with what you have learned? Confident enough for a battle to test your logic and speed? You are in the right place!

About The Event

Each participants are given 5 questions (with a sample input and output) and the candidate has to solve them within the time span of 30 minutes. Each tasks are pretty basic levels and does not require any kind of expertise.

The programming language can be of the candidate's choice. eg: C, C++, Java or Python

Further details of the event will be shared via email or whatsApp to the registered candidates. All the best Folks!
